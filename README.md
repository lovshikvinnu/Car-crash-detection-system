# Smart Vehicle Blackbox with Accident Detection and Emergency Response

## Overview

A low-cost intelligent vehicle blackbox system capable of:

- Accident Detection
- Rollover Detection
- GPS Location Tracking
- Emergency Alert Generation
- Blackbox Event Preservation

## Features

- MPU6050 based accident detection
- GPS location tracking using NEO-6M
- Two-stage emergency alert protocol
- Automatic location update
- Blackbox lock mode after event

## Hardware Used

- Arduino UNO
- MPU6050
- NEO-6M GPS Module
- Breadboard
- Jumper Wires

## Working

1. Monitor vehicle motion continuously.
2. Detect impact or rollover.
3. Trigger emergency alert.
4. Acquire GPS coordinates.
5. Send location update.
6. Lock system until manual reset.
